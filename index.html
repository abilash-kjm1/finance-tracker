One-time setup needed

Connect your free Firebase project so your data can sync across devices:

  1. Go to console.firebase.google.com and create a project (free).
  2. Enable Authentication → Sign-in method → Google.
  3. Create a Cloud Firestore database (production mode).
  4. Project settings → Add a Web app → copy the config.
  5. Paste it into js/firebase.js replacing the placeholder.
